  
<span style="font-family:Arial sans-serif;font-size:16px;">WD Service</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">This example presents the creation of a "service" application with WINDEV.</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">Summary of the example supplied with WINDEV: </span>

<span style="font-family:Arial sans-serif;font-size:14px;">A service is a resident program the can operate outside a Windows session (the application is started before the Windows "Login").</span>

<span style="font-family:Arial sans-serif;font-size:14px;">In most cases, this type of application has no interface (no window)</span>

<span style="font-family:Arial sans-serif;font-size:14px;">For example, a service can be a processing robot (automation of a batch process).</span>

<span style="font-family:Arial sans-serif;font-size:14px;">This example explains how a 'Service' program can be created in WINDEV.</span>
